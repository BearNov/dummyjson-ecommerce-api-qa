# API Test Cases

This folder will contain documented API test cases for the DummyJSON E-Commerce API QA Portfolio project.

The test cases will describe what is being tested, the test data used, the expected result, and the actual result.

## Planned Content

- Product endpoint test cases
- Product search test cases
- Category test cases
- Cart test cases
- User/authentication test cases if suitable
- Positive test cases
- Negative test cases
- Edge case scenarios
- Test execution status
