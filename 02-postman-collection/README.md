# Postman Collection

This folder will contain the Postman collection and environment files for the DummyJSON E-Commerce API QA Portfolio project.

The Postman collection will be used to test selected e-commerce API endpoints such as products, product search, categories, carts, users, and authentication.

## Planned Content

- Postman collection export
- Postman environment export
- API request examples
- Positive API tests
- Negative API tests
- Postman assertions
- Notes for running the collection
