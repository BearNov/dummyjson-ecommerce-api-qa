# API Test Plan

This folder will contain the API test plan for the DummyJSON E-Commerce API QA Portfolio project.

The purpose of the test plan is to define the testing scope, selected API areas, test approach, risks, assumptions, and out-of-scope items before writing detailed test cases.

## Planned Content

- API overview
- Selected endpoints
- Test objectives
- Test scope
- Out-of-scope items
- Test data approach
- Risks and assumptions
- Entry and exit criteria
